Crescendo

Live demo ↗

A gamified RPG productivity platform. Failure has real cost here: streak breaks trigger rank-scaled "Aura" deductions, and a RankUp Ritual system opens a timed task window with a 7-day lockout if you miss it.

Django 5.2DRFReact (Vite)Vanilla CSSGemini APISQLite
  • Decoupled DRF + React architecture with JWT auth, custom user models, and email verification — full session security across a separated client-server setup.
  • Rank-gated penalty engine: streak failures cost rank-scaled Aura; the RankUp Ritual enforces deadlines and lockouts entirely through backend business logic.
  • Archetype theming system (Grimward / Ashborne / Goldveil) — rewrites the UI's glassmorphism palette, neon accents, and animations globally based on user selection, persisted server-side.
  • Gemini API integration for AI-powered quest generation, auto-producing subtask checklists from a quest title, plus a Grand Roadmap system with deadline oaths and GP rewards.

Magic Todo

Live demo ↗

An AI-powered task architect that breaks goals into actionable steps. Built to stay reliable even when the AI provider isn't — rate limits and outages are handled, not ignored.

DRFReact.jsTailwind CSSAxiosGemini APIPostgreSQL
  • Full-stack task manager with JWT-based authentication, a DRF API backend, and a React (Vite) frontend following a clean client-server split.
  • Gemini 2.0 Flash-Lite integration for AI task deconstruction, with a retry mechanism using exponential backoff to handle HTTP 429 rate limits and 503 outages.
  • REST API with full CRUD via DRF serializers and class-based views, CORS configured for cross-origin calls between the Vercel frontend and Render backend.
  • Real-time prioritization and categorization on the frontend — state updates reflect instantly with no full-page reloads.
