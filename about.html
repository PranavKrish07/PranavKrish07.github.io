About

Backend-leaning, full-stack in practice

I'm a full-stack developer who ships complete, decoupled applications — Django REST Framework on the backend, React.js on the frontend, talking to each other over a clean API contract instead of a tangled monolith.

My flagship project, Crescendo, turns productivity into an RPG: streaks cost you "Aura," ranks gate what you can attempt, and a 7-day lockout enforces real consequences for failure — all backed by actual business logic, not just a pretty UI. I followed that with Magic Todo, an AI-assisted task manager built around the same principles: JWT authentication, a proper DRF API, and resilient handling of AI rate limits via exponential backoff.

I care most about the parts of an app that don't show up in a screenshot — session security, API design, and what happens when an external service (like Gemini) returns a 429 instead of a clean response. I'm currently finishing my B.Tech in Computer Science & Engineering at Lovely Professional University, graduating in 2026.
